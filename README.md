# Selenium Automation Project

## Overview
This project automates different UI elements on a test website using Selenium WebDriver with Java.

## Features
- Form input automation
- Radio button and checkbox handling
- Dropdown selection
- Date picker handling (3 types)
- File upload (single and multiple)
- Alerts handling
- Actions class (double click, hover, drag and drop, slider)
- JavaScript execution (scrolling, setting values)

## Tech Stack
- Java
- Selenium WebDriver
- WebDriverManager
- ChromeDriver

## Project Structure
- MainCapstone.java → main automation script

## How to Run
1. Clone the repository
2. Open in Eclipse or IntelliJ
3. Run MainCapstone.java

## Key Learning
- Handling different types of date pickers
- Working with dynamic elements
- Using JavaScriptExecutor in Selenium
- Performing advanced user actions
